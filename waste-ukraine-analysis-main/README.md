# Утворення відходів в Україні по галузях економіки

Аналіз даних Державної служби статистики України та Eurostat за 2015–2024 роки.

## Що досліджувалось
- Які галузі генерують найбільше відходів
- Динаміка змін за 10 років (вплив 2014 і 2022 років)
- Частка небезпечних відходів (клас I–III) по регіонах і галузях
- Структура відходів за матеріалами
- Порівняння вироблення промислових відходів в Україні з країнами Європис зі схожою економікою (Болгарія, Фінляндія)

## Головні висновки
- 86% промислових відходів України формує один сектор — видобуток металевих руд; побутове сміття, що домінує у публічному дискурсі, складає лише 2.3%
- Дніпропетровська область — 114.6 млн тонн (48% від усіх регіонів)
- Фармацевтика має найвищу частку небезпечних відходів — 7.5% від свого обсягу
- Війна кардинально змінила структуру: обсяг відходів від видобутку металів у 2022 році скоротився вдвічі — з 366 до 170 млн тонн
- Україна переробляє реально лише 3–5% придатних для переробки відходів; законодавча база сформована, але фінансування несумірне з масштабом проблеми
- Ринок управління промисловими відходами в Європі — 394 млрд доларів; Україна з її обсягами має значний потенціал, який наразі не реалізується
