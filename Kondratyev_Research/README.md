
# Аналіз надсмертності в Україні (2013-2021): Демографічний ретро-проєкт

Цей проєкт присвячений аналізу причин смертності населення України за період, що передував повномасштабній війні. Дослідження базується на відкритих даних Держстату та візуалізоване за допомогою Power BI.

## Ключові інсайти
* **Гендерна прірва:** Чоловіча смертність стабільно на 30-40% вища за жіночу, особливо у працездатному віці.
* **Демографічний пік:** Найбільша кількість втрат зосереджена у віковій групі 60-69 років, що сумарно становить близько 500 тис. осіб.
* **Зміна структури:** До 2020 року домінувала онкологія, але у 2021 році COVID-19 став причиною №1 за річним показником.

## Методологія (Data Pipeline)
Для забезпечення достовірності даних було проведено:
1. **Python Preprocessing:** Фільтрація лише абсолютних показників ("Кількість померлих") для уникнення змішування з відносними коефіцієнтами.
2. **Data Cleaning:** Усунення дублюючих рядків "Усього" через Power Query та обробка текстових аномалій (Trim/Clean).
3. **Data Modeling:** Трансформація широкої таблиці (стовпці років) у довгу (Unpivoting) для коректної побудови часових рядів.
