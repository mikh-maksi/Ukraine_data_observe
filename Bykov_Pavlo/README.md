# Веб-додаток для агрегації даних

## Про проєкт
Сайт, розроблений для збору та відображення рішень Зміївської ради. Проєкт забезпечує зручний доступ до офіційної документації.

## Основний функціонал
*   **Головна сторінка** Центральна панель з доступом до всіх розділів ресурсу.
*   **Рішення місцевих рад** Окремий розділ для перегляду рішень місцевої ради.

## Структура репозиторію
*   `index.html` — Головна сторінка ресурсу.
*   `decisions.html` — Сторінка з винесеними рішеннями ради.
*   `/css` — Стилі оформлення інтерфейсу (дизайн сайту).
*   `/images` — Графічні матеріали та зображення.

## Використані технології
*   HTML — структура сторінок.
*   CSS — візуальне оформлення.

## Інструкція з перегляду
1. Завантажте файли або скопіюйте репозиторій.
2. Для перегляду сайту запустіть файл `index.html` у вашому браузері.
